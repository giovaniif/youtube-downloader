<b>Make sure you have Node.js installed to run this app</b>

- Clone this repo

```bash
  $ git clone https://github.com/giovaniif/youtube-downloader.git
```

- Run the server

```bash
  $ cd youtube-downloader
  $ cd server
  $ yarn && yarn start
```

- Run the app

```bash
  $ cd youtube-downloader
  $ yarn && yarn dev
```
